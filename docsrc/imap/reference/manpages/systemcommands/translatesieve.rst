==================
**translatesieve**
==================

Script to translate sieve scripts to use unixhierarchysep and/or altnamespace. Make sure you run it as the cyrus user

Synopsis
========

.. parsed-literal::

    **translatesieve** 

Description
===========

translatesieve [-f] [imapd.conf]

Options
=======

.. program:: translatesieve

.. option:: -f

    Keep going on errors.
