Unpacking the Cyrus IMAP Server Software

Download the software, if you haven't already done so.

Uncompress the software. To convert the file to a tar file, type: 

   gunzip cyrus-imapd-v1.1beta.tar.gz
To unpack the distribution, type: 
               
   tar xf cyrus-imapd-v1.1beta.tar

If you type "ls -l", the source directory should look something like this: 

   drwxr-xr-x 10 root     2048 Aug 25 08:05 cyrus-1.2
   -rw-r--r--  1 root  1116160 Aug 24 15:47 cyrus-imapd-v1.2beta.tar
Once you have downloaded and unpacked the files, read the installation document.
